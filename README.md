#  Library Management System (Java OOP)

## Objective
A simple Library Management System built using **Java OOP** concepts:
- Abstraction
- Encapsulation
- Inheritance
- Polymorphism

## Features
- Add books and users
- Issue a book to a user
- Return a book
- Display available/issued books
